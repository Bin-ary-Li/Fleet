Detailed Description

A special weighted reservoir sampling class that allows for logarithmic weights to do this, we use a transformation following https://en.wikipedia.org/wiki/Reservoir_sampling#Weighted_random_sampling_using_Reservoir basically, we want to give a weight that is r^1/w, or log(r)/w, or log(log(r))-log(w). But the problem is that log(r) is negative so log(log(r)) is not defined. Instead, we'll use the function f(x)=-log(-log(x)), which is monotonic. So then, -log(-log(r^1/w)) = -log(-log(r)/w) = -log(-log(r)*1/w) = -[log(-log(r)) - log(w)] = -log(-log(r)) + log(w)

An item in a reservoir sample, grouping together an element x and its log weights, value, etc.
